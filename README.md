# 🌐 Meus Estudos: HTML & CSS

Bem-vindo(a) ao meu repositório de estudos! Criei este espaço para registrar minha evolução no desenvolvimento web Front-End, armazenando todos os exercícios, componentes e projetos que construo enquanto aprendo **HTML5** e **CSS3**.

## 🎯 Objetivo
Documentar meu progresso, testar novas propriedades, praticar layouts responsivos e construir uma base sólida para, no futuro, avançar para outras tecnologias.

## 🛠️ Tecnologias Utilizadas
* **HTML5:** Semântica, acessibilidade e estruturação.
* **CSS3:** Flexbox, Grid Layout, animações, variáveis e responsividade (Media Queries).

## 📂 O que você vai encontrar aqui
Irei adicionar alguns projetos como forma de aprendizagem e consolidação de conhecimentos

## 🚀 Como visualizar os projetos
A maioria dos projetos pode ser visualizada diretamente no navegador. Basta clonar o repositório e abrir o arquivo `index.html` de cada pasta.
